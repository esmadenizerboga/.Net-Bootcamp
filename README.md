# 💻 .NET Bootcamp — Techcareer İşbirliği

Techcareer iş birliğiyle düzenlenen .NET Bootcamp kapsamında geliştirilen çalışmaları içeren repodur. .NET Framework temel kavramları ve web geliştirme uygulamalarını kapsamaktadır.

## 📚 Bootcamp Hakkında

Bu bootcamp sürecinde .NET teknolojileri, C# programlama dili ve web geliştirme konularında uygulamalı çalışmalar gerçekleştirilmiştir. Veritabanı bağlantısı, CRUD operasyonları ve ASP.NET temel yapısı üzerine projeler geliştirilmiştir.

## 🚀 İçerik

- C# ile temel ve orta düzey programlama uygulamaları
- Veritabanı bağlantısı ve CRUD operasyonları
- ASP.NET ile web arayüzü geliştirme
- HTML, CSS ve JavaScript ile frontend entegrasyonu

## 🛠️ Kullanılan Teknolojiler

| Teknoloji | Oran |

| HTML       | %33.5 |
| CSS        | %18.0 |
| JavaScript | %0.9 |

## 💻 Çalıştırma

1. Repoyu klonlayın:
git clone https://github.com/esmadenizerboga/.Net-Bootcamp.git

2. Visual Studio veya Visual Studio Code ile `.sln` dosyasını açın.

3. Gerekli NuGet paketlerini yükleyin ve projeyi çalıştırın.

## 📖 Öğrenilen Konular

- C# sözdizimi ve nesne yönelimli programlama (OOP)
- .NET Framework / .NET Core temelleri
- Entity Framework ile veritabanı işlemleri
- CRUD (Create, Read, Update, Delete) operasyonları
- ASP.NET MVC yapısı
